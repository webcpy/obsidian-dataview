## 支持

您在有问题时会去哪里寻找帮助呢？
这里是我们建议您尝试的方法。

1. 在 GitHub 讨论和 Obsidian 论坛上搜索您的问题。
2. 在 Discord 中搜索可能的解决方案。
3. 根据您的问题复杂性：

- 如果您需要接近同步的通信，请使用 Discord。请注意，我们是一个志愿者社区，可能会有回复延迟。
- 如果您认为您的问题需要几天时间解决，并且需要异步通信，请在 GitHub 上创建一个讨论。

4. 如果您发现了一个错误，请在 [仓库的 issue](*** 中报告。
